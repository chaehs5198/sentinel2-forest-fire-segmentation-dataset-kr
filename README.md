# sentinel2-forest-fire-segmentation-dataset-kr

Manually-annotated Sentinel-2 wildfire burned area segmentation dataset for South Korea (2016–2022).
Covers 303 wildfire events and 353 image patches with binary burned area(including active fire) masks.

Further details on the data and methodology can be found in [Chae & Choi (2024)](https://doi.org/10.22776/kgs.2024.59.2.283), *Journal of the Korean Geographical Society*, 59(2), 283–294. (written in Korean)
